NetgenLayoutsExtendedBundle
===========================
ein Bundle mit eigenen Erweiterung z.B. Blöcken für unsere Installationen


Blöcke
======

Trenner [spacer]
------

Redakteur kann mithilfe von einer horizontalen Linie ( \<hr /> ) oder mit Bildern Blöcke trennen.

@see netgen_layouts_extended.yaml

    netgen_layouts_extended:
        spacer:
            images:
                - { name: '%Bildname%', path: '%Bildpfad%' }
